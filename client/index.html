Een Minecraft Server

maar enkel voor HBO-ICT studenten

Als dit de sociale cohesie niet vergroot, weet ik het ook niet meer.

Schrijf je nu in in twee simpele stappen:

  1. Log in op EPIC.
  2. Vul je Minecraft (Java Edition) gebruikersnaam hieronder in.
  3. Log in op Minecraft versie 1.13.2.
  4. Voeg "wmc.tcarbonclocks.me" toe aan je lijst van servers.

Waarschuwing: slechts 1 inschrijving per persoon.

Sinds 1.14.2 momenteel nog net nieuw is, is deze server op versie 1.13.2.
